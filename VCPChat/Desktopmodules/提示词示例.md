
## 桌面挂件 API（VCPdesktop Widget Sandbox）

当你需要在桌面画布上创建交互式挂件时，使用 <<<[DESKTOP_PUSH]>>> 语法输出 HTML/CSS/JS。
你的脚本在沙箱中执行，以下 API 自动可用：

### document（已代理到 Shadow DOM）
- document.querySelector/querySelectorAll/getElementById → 在 widget 内部查找
- document.createElement/createTextNode → 正常创建元素

### vcpAPI（后端数据访问，自动认证）
- vcpAPI.weather() → 获取天气数据
- vcpAPI.fetch('/admin_api/任意端点') → 通用后端 API（Basic Auth）
- vcpAPI.post(messages, options) → 标准 OpenAI 兼容 chat completion 请求（Bearer Token，安全代理，widget 无需知道 URL 和 API Key）

天气数据结构：
{
  city, hourly: [{fxTime, temp, text, icon, humidity, windDir, windScale}],
  daily: [{fxDate, tempMin, tempMax, textDay, iconDay}],
  airQuality: {aqi, category, pm2p5},
  warning: [{title, text}]
}

### musicAPI（音乐播放器控制）
- musicAPI.getState() → 获取播放状态
  返回: {is_playing, file_path, position_secs, duration_secs, volume, ...}
- musicAPI.play() → 播放
- musicAPI.pause() → 暂停  
- musicAPI.seek(秒数) → 跳转
- musicAPI.setVolume(0-100) → 设置音量
- musicAPI.send('music-remote-command', 'next') → 下一首
- musicAPI.send('music-remote-command', 'previous') → 上一首

vcpAPI.post() 参数说明：
- messages: OpenAI 格式消息数组 [{role: "system", content: "..."}, {role: "user", content: "..."}]
- options.model: 模型名称（如 "gemini-2.5-flash-preview"）
- options.temperature: 温度参数（0-2）
- options.max_tokens: 最大输出 token 数
- 返回: {success, content, usage, raw, error}

### 使用示例

AI 翻译挂件（使用 vcpAPI.post()）：
<<<[DESKTOP_PUSH]>>>
<div style="padding:16px;background:rgba(20,28,58,0.9);color:#fff;border-radius:12px;">
  <textarea id="src" placeholder="输入文本..." style="width:100%;height:60px;background:rgba(255,255,255,0.05);color:#fff;border:1px solid rgba(255,255,255,0.1);border-radius:8px;padding:8px;"></textarea>
  <button id="btn" style="margin-top:8px;padding:6px 16px;background:rgba(60,120,255,0.6);color:#fff;border:none;border-radius:6px;cursor:pointer;">翻译为英文</button>
  <div id="out" style="margin-top:8px;min-height:40px;padding:8px;background:rgba(255,255,255,0.04);border-radius:8px;"></div>
</div>
<script>
document.getElementById('btn').addEventListener('click', async function() {
    var text = document.getElementById('src').value;
    if (!text) return;
    document.getElementById('out').textContent = '翻译中...';
    var result = await vcpAPI.post([
        { role: "system", content: "你是翻译引擎，只输出翻译结果。" },
        { role: "user", content: "翻译为英文：\n\n" + text }
    ], { model: "gemini-2.5-flash-preview", temperature: 0.3 });
    document.getElementById('out').textContent = result.success ? result.content : '错误: ' + result.error;
});
</script>
<<<[DESKTOP_PUSH_END]>>>

天气挂件：

<div id="w" style="padding:16px;background:rgba(0,0,0,0.5);color:#fff;border-radius:12px;">加载中...</div>
<script>
vcpAPI.weather().then(function(data) {
    var h = data.hourly[0];
    document.getElementById('w').innerHTML = '<h2>'+h.temp+'°C '+h.text+'</h2><p>湿度'+h.humidity+'%</p>';

<<<[DESKTOP_PUSH_END]>>>

音乐控制条：
<<<[DESKTOP_PUSH]>>>
<div style="display:flex;gap:8px;padding:12px;background:rgba(20,20,40,0.85);border-radius:20px;color:#fff;">
    <span id="title">--</span>
    <button onclick="musicAPI.send('music-remote-command','previous')">⏮</button>
    <button id="pp" onclick="this.dataset.playing==='true'?musicAPI.pause():musicAPI.play()">▶</button>
    <button onclick="musicAPI.send('music-remote-command','next')">⏭</button>

<script>
setInterval(async function(){
    var s = await musicAPI.getState();
    if(!s) return;
    var pp = document.getElementById('pp');
    pp.textContent = s.is_playing ? '⏸' : '▶';
    pp.dataset.playing = s.is_playing;
    var fp = s.file_path || '';
    var parts = fp.split(/[\\\\/]/);
    var name = parts[parts.length-1] || '--';
    var dot = name.lastIndexOf('.');
    if(dot>0) name = name.substring(0,dot);
    document.getElementById('title').textContent = name;
}, 2000);
</script>
<<<[DESKTOP_PUSH_END]>>>

# 调用工具推送壁纸 
功能: 推送壁纸到用户的VCPdesktop桌面。支持多种壁纸来源：
1. 直接URL（http/https）：推送网络图片或视频作为壁纸。
2. 本地文件URL（file:///...）：推送本地图片、视频或HTML文件作为壁纸。
3. HTML内容：直接传入一段HTML代码，系统会将其保存为HTML文件并作为动态壁纸加载（适合粒子效果、shader动画等）。

壁纸类型会根据文件扩展名或内容自动检测：图片(jpg/png/gif/webp/svg)、视频(mp4/webm)、HTML动态壁纸(html/htm)。

参数:
- wallpaperSource (字符串, 必需): 壁纸来源。可以是：
  - HTTP/HTTPS URL (如 https://example.com/bg.jpg)
  - 本地文件 file:// URL (如 file:///C:/wallpapers/bg.png)
  - 完整的HTML代码（以 <!DOCTYPE 或 <html 开头时会被识别为HTML内容）
调用格式:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」SetWallpaper「末」,
wallpaperSource:「始」https://example.com/beautiful-landscape.jpg「末」
<<<[END_TOOL_REQUEST]>>>

创建HTML动态壁纸示例:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」SetWallpaper「末」,
wallpaperSource:「始」<!DOCTYPE html><html><head><style>body{margin:0;overflow:hidden;background:#000}canvas{display:block}</style></head><body><canvas id="c"></canvas><script>/* 粒子动画代码 */</script></body></html>「末」
<<<[END_TOOL_REQUEST]>>>

# 查询桌面
功能: 感知查询当前VCPdesktop桌面上的所有挂件(widget)和图标(shortcut icon)信息。
返回内容包括：
- widgets: 桌面上所有活跃挂件的ID列表。如果挂件已被收藏，还会显示收藏名称和对应的持久化目录路径（可配合FileOperator插件读写维护收藏的HTML内容）。
- icons: 桌面上所有快捷方式图标的名称列表。

此命令无需额外参数。
调用格式:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」QueryDesktop「末」
<<<[END_TOOL_REQUEST]>>>

# 查询Dock应用列表
功能: 查询VCPdesktop的Dock栏完整应用列表，了解所有可用的应用、工具和启动方式。
返回内容包括：
- Dock快捷方式: 用户导入的Windows桌面快捷方式列表（名称、类型、可见性、启动代码）
- VChat内部应用: 12个硬编码的VChat核心应用（如音乐播放器、笔记、翻译等），始终可用
- Windows系统工具: 5个系统工具快捷入口（显示设置、控制面板等），始终可用
- 内置桌面挂件: 天气挂件、音乐播放条、应用托盘等内置组件

每个应用都附带对应的启动代码示例，可在Widget脚本中直接使用 `window.VCPDesktop.dock.launch(item)` 调用。

此命令无需额外参数。
调用格式:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」QueryDock「末」
<<<[END_TOOL_REQUEST]>>>

# 获取widget源码
功能: 查看桌面上指定挂件(widget)的HTML源码内容。通过提供挂件ID，可以获取该挂件当前正在渲染的完整HTML代码。这对于理解挂件的结构、调试问题或基于现有挂件进行修改非常有用。

建议先调用 QueryDesktop 获取挂件ID列表，再使用此命令查看特定挂件的源码。

参数:
- widgetId (字符串, 必需): 要查看源码的挂件ID。
调用格式:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」ViewWidgetSource「末」,
widgetId:「始」widget-abc123「末」
<<<[END_TOOL_REQUEST]>>>

## 在分布式里创建插件

功能: 在用户的VCPdesktop桌面上创建一个新的挂件(widget)。你可以传入完整的HTML代码，系统会在桌面画布上渲染该挂件。支持指定初始位置、尺寸，并可选择是否自动收藏。

这是最灵活的桌面交互方式，适合创建：天气挂件、时钟、待办事项、数据可视化面板、小游戏、动画效果等任何基于HTML的交互式内容。

挂件运行在Shadow DOM沙箱中，脚本通过安全包装执行。挂件内可以使用以下API：
- `document.querySelector/getElementById` 等 DOM 操作（作用域限定在挂件内部）
- `vcpAPI.fetch(endpoint, opts)` - 调用VCP后端API
- `vcpAPI.weather()` - 获取天气数据
- `musicAPI.play/pause/getState/setVolume/seek` - 控制音乐播放

参数:
- htmlContent (字符串, 必需): 挂件的完整HTML代码。可以包含 <style> 和 <script> 标签。
- x (数字, 可选): 挂件左上角的X坐标（像素），默认100。
- y (数字, 可选): 挂件左上角的Y坐标（像素），默认100。
- width (数字, 可选): 挂件初始宽度（像素），默认320。挂件会根据内容自动调整。
- height (数字, 可选): 挂件初始高度（像素），默认200。挂件会根据内容自动调整。
- widgetId (字符串, 可选): 自定义挂件ID。默认自动生成。
- autoSave (布尔, 可选): 是否自动将挂件收藏到侧栏，默认false。
- saveName (字符串, 可选): 收藏名称（仅当autoSave为true时有效）。

调用示例 - 创建一个简单时钟挂件:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」CreateWidget「末」,
htmlContent:「始」<div style="font-family:monospace;font-size:48px;color:#0ff;text-align:center;padding:20px;background:#111;border-radius:12px"><span id="clock">00:00:00</span></div><script>setInterval(()=>{const d=new Date();document.getElementById('clock').textContent=d.toLocaleTimeString()},1000)</script>「末」,
x:「始」200「末」,
y:「始」150「末」
<<<[END_TOOL_REQUEST]>>>

调用示例 - 创建并自动收藏:
<<<[TOOL_REQUEST]>>>
tool_name:「始」DesktopRemote「末」,
command:「始」CreateWidget「末」,
htmlContent:「始」<div style="padding:16px;background:linear-gradient(135deg,#667eea,#764ba2);color:#fff;border-radius:12px;font-size:18px">Hello Desktop!</div>「末」,
autoSave:「始」true「末」,
saveName:「始」问候挂件「末」
<<<[END_TOOL_REQUEST]>>>
