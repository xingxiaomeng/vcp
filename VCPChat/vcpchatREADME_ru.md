# VCPChat — Распределенный полностековый движок на базе ИИ

[简体中文](README.md) | [English](vcpchatREADME_en.md) | [日本語](vcpchatREADME_jp.md)

**VCPChat — это больше, чем просто чат-клиент; это распределенный полностековый базовый движок, стирающий границы между ИИ, интерфейсом (UI) и приложениями.**

Благодаря глубокой виртуализации и инкапсуляции фронтенд-IPC и бэкенд-API, VCPChat создает нативную среду выполнения ИИ, выходящую за рамки традиционных парадигм взаимодействия.

## 🚀 Абстракция уровня движка: чудо в 30 строках кода

Основная ценность VCPChat заключается в его исключительных **возможностях абстракции и интеграции**.

*   **Максимальная инкапсуляция**: Благодаря низкоуровневому потоковому рендерингу, распределенной связи и полностековой системе плагинов, разработчики теперь могут создать полнофункциональный `VChatMini` всего за **30 строк кода**.
*   **Превосходство в измерениях**: По сравнению с 300 000 строк базового кода самого VCPChat, такой высокий уровень абстракции делает разработку сложных ИИ-приложений такой же простой, как сборка конструктора.
*   **Полностековая интеграция**: Будь то аудиодвижок HIFI, потоковый рендеринг HTML или распределенное отслеживание файлов между узлами — все возможности открыты для разработчиков через виртуализированные интерфейсы (vcpAPI).

## Установка и развертывание

1.  **Клонирование репозитория**

    Клонируйте этот проект на свой локальный компьютер:
    ```bash
    git clone https://github.com/lioensky/VCPChat.git
    cd VCPChat
    ```

2.  **Установка зависимостей**

    Для этого проекта требуются среды Node.js и Python.

    *   **Установка зависимостей Node.js:**
        ```bash
        npm install
        ```

    *   **Установка зависимостей Python** (для аудиодвижка, продвинутых плагинов и т. д.):
        ```bash
        pip install -r requirements.txt
        ```

3.  **Запуск приложения**

    *   **Обычный запуск:**
        ```bash
        npm start
        ```

    *   **Тихий запуск (опционально):**
        Вы также можете использовать скрипт `run_silent.vbs` для запуска без окна консоли.

## Глубокая синергия фронтенда и бэкенда: раскрытие бесконечного потенциала ИИ

VCPChat — это не просто интерфейс чата, это «глаза» и «холст» мощной экосистемы бэкенда VCP (VCPToolBox). Глубокая интеграция обоих компонентов направлена на раскрытие бесконечного потенциала ИИ:

*   **Рендеринг продвинутых возможностей**: VCPChat разработан специально для рендеринга сложных выходных данных протокола VCP. Будь то «дневник» активной записи и рефлексии ИИ или мультимедийный контент — всё может быть представлено максимально интуитивно. Пузыри вывода агентов VCPChat обладают почти чудовищными возможностями рендеринга, поддерживая практически все основные виды анимации и документов.
*   **Тяжелые асинхронные задачи**: Бэкенд может выполнять ресурсоемкие задачи, такие как генерация видео или анализ данных. После того как ИИ инициирует задачу, он может немедленно ответить пользователю. По завершении задачи VCPChat через бэкенд-пуш отобразит результаты (например, сгенерированное видео) в потоке диалога в реальном времени, обеспечивая плавный процесс без задержек.
*   **Роевой интеллект агентов**: Бэкенд поддерживает совместную работу нескольких ИИ-агентов, вплоть до того, что агент может активно назначать задачи субагентам. Режим группового чата VCPChat и четкая система маркировки говорящих обеспечивают идеальный интерфейс взаимодействия для этого продвинутого процесса совместной работы «отряда ИИ-горничных».
*   **Богатое мультимедийное взаимодействие**: ИИ может вызывать инструменты VCP через бэкенд для отправки стикеров, воспроизведения музыки и показа видео в диалогах. VCPChat обладает мощным движком рендеринга мультимедиа, продвинутой анимацией пузырей окон и потоковой анимацией, гарантируя максимальное качество аудиовизуального опыта.
*   **Основные мультимодальные возможности VCP**: Это позволяет ИИ действительно видеть и слышать мультимедиа, созданное такими плагинами, как FluxGen или SunoGen.
    *   **Прямая передача Base64**: Позволяет ИИ напрямую вводить данные Base64 в поле `tool`, что значительно упрощает мгновенный вызов мультимедийного контента.
    *   **Глобальный файловый API (`VCPFileAPI` v4.0 Hyper-Stack Tracking)**: Реализует революционное отслеживание гиперстека по полным URL. Теперь, когда ИИ отправляет путь к локальному файлу (например, `H:\MCP\123.txt`) на любом распределенном узле, главный сервер может интеллектуально проанализировать его источник и автоматически запросить данные Base64 файла у исходного узла, обеспечивая бесшовный вызов файлов между серверами.
    *   **Кросс-модальный интеллектуальный перевод**: Реализует «передачу возможностей» от моделей высокого порядка к моделям низкого порядка. Например, модель, способная распознавать аудио, может помочь текстовой модели, интеллектуально переводя аудиоданные Base64, которые та не может обработать, в текстовые описания.
    *   **Распределенная мультимодальная передача (обновление ядра v4.0)**: Благодаря **отслеживанию гиперстека по полным URL**, ИИ на любом узле может напрямую использовать пути к локальным файлам для межсерверных вызовов. Главный сервер автоматически отследит и подтянет данные, полностью устраняя изоляцию файлов в распределенных сетях.
    *   **Интеллектуальная маршрутизация ответов**: Ядро VCP может интеллектуально определять, возвращает ли плагин традиционную текстовую информацию stdio или структурированные данные, содержащие Base64, и автоматически выбирать правильный канал для пересылки и обработки.
    *   **Совместное использование несколькими агентами**: В задачах совместной работы нескольких агентов реализуется интеллектуальный обмен данными Base64, которые при необходимости могут быть динамически преобразованы во временные `fileurl` для доступа других агентов или фронтенд-приложений.

### 🖥️ VCPDesktop — первый в мире ИИ-нативный рабочий стол (22.03.2026)

Появилась по-настоящему революционная функция следующего поколения — **VCPDesktop**. Эта подсистема почти полностью опирается на мощную инфраструктуру VCP и превосходную производительность VChat, полностью стирая границы между ИИ и интерфейсом.

*   **Потоковая передача на рабочий стол**: ИИ может использовать синтаксис `<<<[DESKTOP_PUSH]>>>` для потоковой передачи любого контента на рабочий стол VCP, создавая виджеты в реальном времени. От погоды и расписания до новостей — ИИ может бесшовно создавать, воспринимать, редактировать и управлять этими плавающими виджетами с задержкой на уровне **100 мс**.
*   **Глубокая интеграция с бэкендом**: VCPDesktop глубоко интегрирован с бэкенд-WebSocket и Admin API, обладая богатыми функциями плавающих уведомлений, системного взаимодействия и автоматического подтягивания данных бэкенда.
*   **Персистентность уровня рабочего стола**: Пользователи могут сохранять любые div-пузыри или плавающий HTML, отправленные агентами, для создания постоянных компонентов рабочего стола — включая, помимо прочего, виджеты погоды, музыкальные плееры, веб-виджеты и даже мини-игры.
*   **Глубокая интеграция VCPCli**: Автоматическое получение данных рабочего стола или папок ПК пользователя для синхронизации ярлыков в реальном времени, работая как настоящий рабочий стол.
*   **Движок потокового рендеринга**: Опираясь на движок потокового рендеринга VChat, он может транслировать любой контент, имеет механизм заморозки «надгробий» и обеспечивает производительность, значительно превосходящую обычное ПО на Electron.
*   **Единая глобальная тема**: Поддержка глобального распределенного вещателя тем VCP, обеспечивающего единый стиль UI/UX, позволяя агентам динамически редактировать макеты рабочего стола, стили CSS, обои и продвинутые динамические обои.

## Основные функции

*   **Интеграция с сервером VCP**: Основная функциональность взаимодействия движка зависит от сервера VCP. Он взаимодействует с сервером VCP через HTTP(S), отправляя сообщения пользователя и получая ответы ИИ, поддерживая потоковую передачу для взаимодействия в реальном времени.
*   **Вызов инструментов VCP**: Полная поддержка различных вызовов инструментов, определенных сервером VCP, включая **синхронные инструменты**, требующие немедленного результата (например, вычисления, запросы), и **асинхронные инструменты**, которые могут выполняться в фоновом режиме (например, генерация видео, парсинг длинных текстов с сайтов). VChat глубоко оптимизировал процесс вызова инструментов:
    *   **Интеллектуальные пузыри взаимодействия**: Пузыри вызова инструментов спроектированы так, чтобы автоматически разворачиваться при наведении курсора, четко отображая полный набор инструкций, инициированных ИИ.
    *   **Разнообразные механизмы обратной связи**: После завершения задачи результаты могут быть сообщены пользователю несколькими способами:
        *   **Уведомления WebSocket в реальном времени**: Для сценариев, требующих мгновенной обратной связи.
        *   **Контекстные встроенные пузыри**: Бесшовная интеграция результатов в поток диалога.
        *   **Системные пуш-уведомления на несколько устройств**: Напоминания о завершении важных задач, даже если пользователь не в приложении.
    *   **Совместная оптимизация инструкций**: Перед выполнением вызова инструмента ИИ может активно запрашивать мнение пользователя или приглашать пользователей и других агентов к совместному редактированию и совершенствованию инструкций.
    *   **Надежный механизм прерывания**: Когда пользователь прерывает ответ ИИ, система синхронно прерывает цепочку вызовов инструментов VCP и полностью завершает связанные фоновые процессы.
*   **Пользовательский интерфейс вызова инструментов VCP**:
    *   Мощные инструменты VCP теперь доступны не только ИИ. VChat предоставляет пользователям полный и интуитивно понятный графический интерфейс (GUI) для легкого вызова и выполнения инструментов VCP.
    *   **Команды не требуются**: Пользователям не нужно запоминать и вводить сложные команды; достаточно выбрать инструмент и нажать кнопки пресетов параметров в GUI.
*   **Панель плагина ComfyGen**: Предоставляет богатые панели управления и настройки для ComfyGen — мощного плагина генерации изображений. Это включает в себя детальное управление рабочими процессами (Workflows), моделями LoRA и другими файлами моделей. Кроме того, интегрирован интерфейс управления, похожий на Stable Diffusion WebUI.

*   **Рендеринг дневников VCP**: Возможность рендеринга и отображения содержимого дневников VCP. Это не просто просмотр логов, а окно в то, как ИИ формирует долгосрочную память и самосовершенствуется.
*   **Управление агентами**:
    *   Создание, удаление и настройка нескольких ИИ-агентов.
    *   Установка имени, системного промпта и параметров модели (температура, лимит токенов контекста, макс. токены вывода) для каждого агента.
    *   Управление аватарами агентов.
    *   **Автономное управление темами**: Агенты теперь могут воспринимать, редактировать, изменять и создавать свои собственные списки тем чата. Они могут понимать историю чата, читать конкретные записи или создавать новую тему. Это полезно, когда агент работает в фоновом режиме и ему нужно активно инициировать чат с пользователем.
    *   Поддержка нескольких независимых тем чата (Topics) для каждого агента, включая создание, удаление, переименование, сортировку и **экспорт** (в форматах Markdown или HTML).
*   **Продвинутое управление контекстом (совместимо с SillyTavern)**: VChat реализует механизм управления контекстом, высокосовместимый с SillyTavern, обеспечивая мощную поддержку для детальных и многократно используемых настроек фона диалога.
    *  **Пресеты, карты персонажей и книги мира**: Система VCP теперь полностью поддерживает и позволяет подключать `Пресеты (Preset)`, `Карты персонажей (Character Card)` и `Книги мира (World Book)` формата SillyTavern.
    *  **Визуальный редактор пресетов**: Встроенный мощный визуальный редактор позволяет создавать и редактировать пресеты контекста. Поддерживает сложные правила вставки, такие как `Глубокая вставка (Deep Injection)` и `Относительная вставка (Relative Injection)`.
    *  **Сортировка контекста перетаскиванием**: В интерфейсе чата все вставленные контексты (системные промпты, настройки персонажа, информация о мире и т. д.) четко видны и поддерживают изменение порядка через `drag-and-drop`.
*   **Режим группового чата (Agent Groups)**:
    *   Позволяет нескольким настроенным агентам сотрудничать или участвовать в ролевой игре в одной сессии чата.
    *   Поддержка создания, настройки и управления группами агентов, включая установку имен групп и аватаров.
    *   **Режимы выступления**:
        *   **Последовательный (`sequential`)**: Участники говорят по очереди в заданном порядке.
        *   **Естественный случайный (`naturerandom`)**: Интеллектуально определяет, кто из участников ответит, на основе `@имени_персонажа`, тегов или ключевых слов в сообщении.
        *   **Только по приглашению (`inviteonly`)**: Решение о том, кто говорит, принимает пользователь, нажимая на кнопку агента.
    *   **Групповой промпт (`groupPrompt`)**: Общий фон, правила или системные инструкции для всего группового чата.
    *   **Промпт приглашения (`invitePrompt`)**: Шаблонная строка, используемая системой для предложения конкретному агенту высказаться.
    *   **Система маркировки говорящих**: Система автоматически добавляет маркер говорящего перед каждым сообщением, обычно в формате `[Имя говорящего говорит]: Текст сообщения`.
*   **Групповые файлы / Рабочая область**: Выделенное общее пространство для файлов и рабочая область для каждой группы.
    *   **Централизованное хранение**: Все файлы, связанные с задачами группы, могут быть загружены и сохранены здесь.
    *   **Совместное редактирование**: Поддержка совместного редактирования файлов в рабочей области в реальном времени.
*   **Кросс-терминальная память и бесшовная синхронизация**: Система памяти VChat сосредоточена на бэкенде VCP, создавая единую персистентную библиотеку памяти агентов. Это означает, что независимо от того, какой фронтенд вы используете, вся история диалогов и знания будут синхронизироваться в реальном времени.
*   **Режим Flow Lock**:
    *   **Фокусированное взаимодействие**: При включении этой функции для конкретной темы пользователь временно не может переключать агентов или темы, окно блокируется для обеспечения глубины диалога.
    *   **Проактивность ИИ**: В этом режиме ИИ не просто ждет ввода пользователя, а может активно инициировать диалог, продолжать задачи или сообщать о прогрессе.
*   **Регулярные выражения агентов**: Мощная функциональность регулярных выражений для глубокого контроля поведения агентов в различных сценариях.
*   **VCP Human Toolbox**: Автоматическая генерация графического интерфейса (GUI) для всех плагинов VCP на сервере, что значительно упрощает работу человека с инструментами.
*   **Модуль совместной работы Canvas: интерактивная рабочая область для кода и документов**:
    *   Революционное пространство для совместной работы в реальном времени. Это не просто текстовый редактор, а «живой документ» и «интерактивная доска» с полной средой разработки и рендеринга.
    *   **Встроенная полнофункциональная IDE**: Безопасная песочница для компиляции и выполнения кода (JS, Python, HTML) с отображением результатов в реальном времени.
    *   **Интеграция с супер-рендерером VChat**: Код внутри Canvas (например, HTML, диаграммы Mermaid, визуализация данных Python) может напрямую использовать движок рендеринга VChat для предпросмотра.
*   **Интерфейс чата**:
    *   Поддержка 21 типа рендереров, включая Markdown, Katex, Html, Mermaid, Three.js и др.
    *   **Мощная работа с мультимедиа и файлами**: Поддержка вложений через выбор файлов, вставку из буфера обмена и drag-and-drop.
    *   **Потоковый рендеринг элементов DIV**: Инновационный механизм потокового рендеринга для сложных пузырей сообщений, созданных ИИ, обеспечивающий плавное отображение вложенного контента (например, кода Python внутри таблиц Markdown).
    *   **Пересылка сообщений между чатами**: Полное сохранение контента при пересылке сообщений любому другому агенту или в группу.
    *   **Комментарии к пузырям**: Возможность добавлять комментарии к сообщениям, которые отображаются под оригинальным пузырем.
    *   **Продвинутый режим чтения**: Иммерсивный опыт чтения длинных текстов с поддержкой интерактивных блоков кода (запуск Python через Pyodide WASM, предпросмотр HTML и 3D Three.js).
*   **Революционный дифференциальный рендеринг в реальном времени**: VCPChat внедряет «потоковый дифференциальный рендерер истории чата». При любых изменениях в истории чата интерфейс не обновляется целиком, а «хирургически» обновляет только изменившиеся части, что позволяет ИИ динамически редактировать свои уже сказанные слова в процессе вывода.

## Исправитель URL-адресов ИИ-стикеров

VChat теперь включает интеллектуальный исправитель URL-адресов стикеров, предназначенный для устранения ошибок в ссылках (IP, порты, пути), возникающих из-за галлюцинаций моделей ИИ.

### Как настроить

Для включения этой функции необходимо синхронизировать кэш списка стикеров из проекта VCPToolbox в папку `AppData/generated_lists` проекта VCPChat и настроить пароль в файле `config.env`.

*   **Менеджер данных пользователя (VchatManager)**: Визуальный инструмент на базе Electron для просмотра и редактирования истории чатов, вложений и данных в `AppData`.
*   **Глобальный поиск по чатам**: Мощный поиск (`Ctrl+F`) по всем агентам, группам и темам с возможностью мгновенного перехода к сообщению.
*   **Модуль перевода**: Независимое окно для быстрого перевода между всеми языками с поддержкой пользовательских форматов вывода.
*   **Модуль заметок**: Управление заметками в форматах TXT, MD, PDF и др. с поддержкой Markdown, LaTeX и синхронизацией с базой знаний ИИ.
*   **Профессиональный аудиодвижок**:
    *   Собственное ядро декодирования и алгоритмы обработки сигналов (FIR-EQ, динамическая компенсация).
    *   Поддержка Hi-Res аудио через WASAPI Exclusive и аппаратное декодирование DSD.
    *   Использование Rust для высокопроизводительного ресамплинга с двойной точностью (f64).
*   **Музыкальный плеер**: Поддержка локальных файлов, умный движок лирики (автозагрузка и создание лирики с помощью ИИ) и управление плеером через агентов ИИ.
*   **Интеграция с файловой системой**: Агенты имеют права на чтение и запись в доверенных директориях, могут искать файлы по всему компьютеру (через EverythingAPI) и обрабатывать мультимедиа (извлечение аудио из видео и т. д.).
*   **Супер-управление браузером**: Превращает любую вкладку браузера в интерактивный Markdown-документ, позволяя ИИ «видеть» веб-страницы и управлять ими (клики, ввод текста) с помощью простого синтаксиса.
*   **Осведомленность о системе и семантическое управление рабочим столом**: Позволяет ИИ воспринимать запущенные процессы и окна и управлять компьютером (клики, ввод с клавиатуры) с помощью семантических команд без необходимости точных координат.
*   **Продвинутый плагин редактора потокового мультимедиа**: Позволяет ИИ делать скриншоты окон, редактировать изображения (аннотации, фильтры), выполнять семантическое редактирование (перенос стиля, синтез элементов) и базовый монтаж видео/аудио.
*   **VchatCLI**: Встроенный полнофункциональный терминал, совместимый с PowerShell и WSL, с поддержкой сохранения состояния сессии и безопасного повышения прав.
*   **Плагин супер-костей**: Реалистичные 3D физические кости (d4-d100) с поддержкой тем и «физической магии» со стороны ИИ.
*   **Плагин V-Daily**: Автоматическая редакция новостей ИИ, создающая персонализированные интерактивные ежедневные газеты на основе сканирования мировых СМИ.
*   **Плагин гадания на Таро**: Сложный движок предсказаний, основанный на реальных астрономических данных (положения планет), погодных условиях и состоянии пользователя, исключающий случайные числа.
*   **Flash DeepSearch**: Плагин для быстрого (до 2 минут) создания академических работ с цитатами на основе глубокого поиска и анализа ИИ.
*   **Интеграция с умным домом Mijia**: Позволяет агентам управлять устройствами Xiaomi через естественный язык и автоматизировать сценарии.
*   **Голосовой чат и TTS**: Реальное голосовое взаимодействие с использованием Puppeteer для распознавания речи и интеграцией GPT-SoVITS для высококачественного озвучивания ответов (включая уникальный движок смешанного двуязычного чтения).
