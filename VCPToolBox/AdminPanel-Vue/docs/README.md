## 🚀 阅读建议

### 如果你是新加入的开发者

建议按以下顺序阅读：

1. [项目介绍](../README.md) - 了解项目概况
2. [组合式函数](../src/composables/README.md) - 了解可复用逻辑

### 如果你需要开发新功能

1. 查看 `src/api/` - 确认需要调用的接口
2. 查看 `src/views/` - 参考现有页面实现
3. 查看 `src/composables/README.md` - 了解可复用逻辑

### 如果你需要排查问题

- 查看 `src/platform/` - 认证、消息提示相关问题
- 查看 `src/api/` - 接口调用相关问题
- 查看 `src/router/` - 路由跳转相关问题
- 查看 `src/stores/` - 状态相关问题
