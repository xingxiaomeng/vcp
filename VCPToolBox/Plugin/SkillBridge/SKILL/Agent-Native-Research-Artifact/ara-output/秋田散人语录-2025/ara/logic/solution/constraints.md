# Constraints

## 本文集的局限性

- **L01 — 单一视角**: 本文集仅代表秋田散人个人观点与经验，其路径依赖性强，不一定适用于所有市场参与者。
- **L02 — 时间有效性**: 加密市场变化极快，许多具体套利路径和机会窗口具有时效性，2023年的有效策略在2025年可能已经失效。
- **L03 — 门槛限制**: 许多策略依赖技术能力（爬虫、监控脚本、自动化系统），非技术背景人士难以直接复现。
- **L04 — 风险隐蔽**: 无人区套利的高收益伴随极高风险——一旦被埋，可能连尸骨都找不到。文集虽有提及但未充分展开风险面。
- **L05 — 合规灰色地带**: 跨境资金流动、全球套利路径涉及多地监管，存在合规风险。文集未深入讨论法律层面。
- **L06 — 样本偏差**: 成功案例的自述天然存在幸存者偏差——因同样方式失败的参与者不会出现在文本中。

## 边界条件

- 本文的策略主要适用于有编码能力、愿意投入时间搭建工具的技术人
- 主张主要适用于加密市场，在传统金融市场可能不完全适用
- "不对称战役"的哲学适用于创业和交易，但不一定适用于职业生涯规划
- 逐仓哲学适用于交易，但在资产配置中可能过于保守

## 核心假设

- A01: 加密市场将持续存在定价无效
- A02: 编码能力可以转化为套利优势
- A03: 信息优势可以系统建设
- A04: 逐仓优于全仓
- A05: 不对称战役优于公平竞争
