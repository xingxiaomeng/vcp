# Claims

## C01: 加密世界是技术人历史上直接参与自由贸易的最佳窗口
- **Statement**: 技术人凭借编码能力，可以以前所未有的低成本跨越当地货币和权力边界，直接参与全球资源调配与定价修正。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在 Stack Overflow 年度开发者调查中，从事加密/区块链相关工作的开发者所报告的中位收入，低于从事传统金融软件开发的中位收入。
- **Proof**: [E01, E02]

## C02: 套利是对跨经济系统错误定价的修正，而非长期经营
- **Statement**: 套利赚的是市场中"错配"和"迟缓"的钱，不是别人的长期经营成果。同一标的在尚未充分连接的不同经济系统间会产生极大的定价差异。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在任意两个具有相同交易对和流动性的 CEX 之间，持续存在超过 0.5% 的价差且持续时间超过 24 小时（排除网络拥堵或充提关闭期间）。
- **Proof**: [E03, E04, E05]

## C03: 信息工具区分人群——监听线路的广度决定获利资格
- **Statement**: 套利者必须在广泛且个性化的信源组合上建立监听系统。要么监听线路只有你一人，要么你是反应最迅速的人。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 一个仅使用公开推特 feed 和主流新闻 API 的交易者，在三个月内的平均套利收益率不低于使用自建爬虫和冷门信源的交易者。
- **Proof**: [E06, E07]

## C04: 真正赚钱的机会藏在"无人区"——有技术门槛的冷门协议
- **Statement**: 存在大量协议和流动性池几乎无人监听，因为存在技术门槛。一旦币价波动打乱价格锚，到处都是利润空间。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 对排名前 100 的 DeFi 协议进行系统性扫描，发现所有协议的链上价格与外部 CEX 价格的偏差均在 0.1% 以内，无任何可维持超过 5 分钟的套利空间。
- **Proof**: [E06]

## C05: 跨系统套利思维优于单一系统内的竞争
- **Statement**: 同一经济系统内部交易摩擦低传导快，套利空间有限。跨链、Cex-Dex、跨境、跨行业、跨周期的贸易思维才能产生持续优势。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 一个仅专注于单一链上 DEX 套利的交易者，其年度风险调整后回报率不低于同时参与跨链、CEX-DEX、跨境套利的交易者。
- **Proof**: [E03, E05]

## C06: 逐仓是优于全仓的风险管理哲学
- **Statement**: 核心是控制单一仓位的保证金大小，控制你在同一件事上犯错的代价。爆仓实质是止损，是帮你把问题控制在局部。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在相同交易策略和资金规模下，连续 12 个月的全仓模式回撤比例不高于逐仓模式，或其风险调整后夏普比率不低。
- **Proof**: [E08]

## C07: 区块链预测市场有望成为极具影响力的真相机器
- **Statement**: 预测市场将真相变为可定价可交易的商品，把知识嵌入价格信号。参与越充分，盘口价格越逼近真实。Polymarket 等平台展示了这一方向的巨大潜力。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在一年以上的运行周期中，Polymarket 的主要盘口结算价格与实际结果的一致率低于 90%，或显著低于专业预测机构的准确率。
- **Proof**: [E09]

## C08: 编码即权力——工程师是天然企业主
- **Statement**: 具有互联网编码能力和传媒影响力的人，即便没有资本，也是天然的企业主。他们可以以极低成本跨越当地货币与权力，对社会资源进行调度与编码。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 一项针对技术创业者的调查显示，仅有编码能力而无资本积累的创业者，其五年存活率低于同等资本水平的非技术创业者。
- **Proof**: [E01, E02]

## C09: 只打不对称战役，避开公平对称竞争
- **Statement**: 创业者不应在任何公平游戏上投入过多精力，只玩信息碾压局，不玩势均力敌的公平对称局。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在信息完全对称的市场环境（如主流加密货币现货交易）中，持续投入专业训练的个人交易者年化收益率可以稳定超过专业机构交易者的平均水平。
- **Proof**: [E10]

## C10: 程序即市场纪律——编码是对市场的假设底线和敬畏
- **Statement**: 每一行 if/else 代表对市场的一条假设，每一个 risk limit 代表能承受的最坏情况，每一个 alert 代表对不确定性的敬畏。把自己的行为原则写进程序里。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 手工交易者可以在连续六个月中，在相同回撤限制下达到与程序化交易同等的收益率和执行一致性。
- **Proof**: [E07]

## C11: 财富教育应在年轻时完成——二十岁的试错成本远低于四十岁
- **Statement**: 普通人家庭出身的年轻人，早年赚到的远超同龄人的钱本质上是见世面的钱，花在自我教育和心理建设上比存起来更有价值。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 一项对 35 岁以上创业者的横截面调查显示：年轻时经历重大财务损失的人，其 40 岁时的净资产中位数不显著高于同龄人中未经历此类损失的人。
- **Proof**: [E11]

## C12: 真正的企业家倾向于高周转率经营而非高杠杆金融
- **Statement**: 用高周转率与执行力创造价值是控制确定性，用利息杠杆年化赌概率是在赌运气。前者更具韧性，后者更敏感于黑天鹅。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在 2018 年和 2022 年两次加密熊市中，采用高杠杆策略的基金的平均回撤幅度小于采用高周转率低杠杆策略的基金。
- **Proof**: [E08]

## C13: 欲望是从零到一的朋友，从一到一百的敌人
- **Statement**: 心性普通的人一次成功就会停止前进。许多能力足够的人跨不过原始积累阶段，是因为欲望还不够。跨过之后需要克制欲望才能继续前进。
- **Status**: hypothesis
- **Provenance**: user
- **Falsification criteria**: 在原始积累阶段之后，维持同样的欲望水平仍能带来持续成长。
- **Proof**: [E10]

## C14: 信息革命使传统统治手段受挫——多信源时代下信息愚昧难以覆盖新生代
- **Statement**: 信息公开导致真相被定价、需求被发现、撮合被放大。公众更愿意追逐真相，因为追真相能带来经济回报。
- **Status**: hypothesis
- **Provenance**: user
- **Falsification criteria**: 在信息充分流通的环境下，旧式统治方法仍能有效维持其效力不衰减。
- **Proof**: [E12]

## C15: 最好的市场复盘是编写更多父类与公共模块
- **Statement**: 技术人除了写复盘文章，更重要的是从工业场景中抽象出共性，代码重构成可复用模块、接口设计成跨系统兼容。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在开源社区中，撰写技术复盘文章获得的 star 数和 fork 数，持续高于发布具有高抽象程度的公共模块库。
- **Proof**: [E01]

## C16: 搬砖人是市场的铲子——通过修正无效定价和传递流动性获利
- **Statement**: 搬砖人表面上是 taker 吃单者，本质上是帮助市场联通的信息 maker。获利来源是市场的财富效应——财富效应越强，流动性的单侧定价越无效。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在市场财富效应高峰时期，搬砖人的平均利润率不高于市场处于平静时期的利润率。
- **Proof**: [E13]

## C17: 商业教育本质上是自我主权教育——天然被主流渠道排斥
- **Statement**: 一切与资本主义解剖相关的商业教育内容——成本模型、利润模型、博弈方案、贸易与投机原理——在某些环境中是天然被排斥的。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 一项对全球排名前 50 的经济学本科课程的分析显示，超过 80% 的必修课程涵盖了私人成本模型、博弈论和贸易理论的完整教学。
- **Proof**: [E14]

## C18: 有效合作的前提是承认我不懂——给团队留下补位空间
- **Statement**: 能坦然说出我不懂是成熟的标志，意味着认识到个人认知边界、为合作创造土壤。故作全知全能是最糟糕的合作姿态。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在高度复杂的现代技术项目中，单一个体可以独立完成所有关键工作而不需要团队协作。
- **Proof**: [E15]

## C19: 起步阶段深度优先，俱乐部内广度优先
- **Statement**: 对起步阶段深度更重要，是进入高手俱乐部的门票。对俱乐部内的游戏广度更关键，需要比别人更擅长发掘新路径。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 在职业起步阶段，广泛涉猎多个领域比在单一领域深耕带来更高的长期回报。
- **Proof**: [E14]

## C20: 从运维到企业主——编码能力的演进即资源编码能力的跃迁
- **Statement**: 建立企业和品牌是更高维度的系统编码。企业是跨主体的程序：代码是执行层，制度是约束层，文化是粘合层，品牌是对外 API。
- **Status**: supported
- **Provenance**: user
- **Falsification criteria**: 编码能力只能用于软件开发和自动化，无法迁移到企业制度设计、品牌建设和文化塑造等更高维度。
- **Proof**: [E02, E15]
