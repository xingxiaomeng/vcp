# Concepts

## 套利 (Arbitrage)
- **定义**: 纠正错误定价、修正资源错配的过程。赚的是市场中"错配"和"迟缓"的钱，不是别人的长期经营成果。
- **本质**: 守株待兔的生意——大部分时间没有机会，但一旦出现特殊时点，赚钱效率极端可观。
- **源**: 秋田散人语录, §4
- **关联主张**: C02, C04

## 跨系统贸易思维 (Cross-System Trade Thinking)
- **定义**: 套利真正有意思的地方——同一标的在不同经济系统间存在极大定价差异，源于（1）流通进度差异、（2）底层tick频率不同、（3）市场投票自由度不同。
- **五种系统分界**: 跨链、DEX与CEX之间、跨境、跨行业、跨周期。
- **源**: 秋田散人语录, §6
- **关联主张**: C05

## 无人区 (No Man's Land)
- **定义**: 有技术门槛、几乎没人监听的协议/流动性池。一旦币价波动打乱价格锚，到处都是利润空间。获利份额永远是100%。
- **源**: 秋田散人语录, §5
- **关联主张**: C04

## 信息买办 (Information Broker)
- **定义**: 利用信息通道优势，将一手信息转化为市场定价优势的角色。核心能力是信源广度、过滤精度和反应速度。
- **源**: 秋田散人语录, §9, §11
- **关联主张**: C03

## 逐仓 (Isolated Margin)
- **定义**: 一种交易纪律——控制单一仓位的保证金大小，让每一笔交易的亏损始终在可接受范围内。爆仓实质等于止损。
- **核心**: 有限损失等于有限回撤等于心态稳定等于降低情绪化决策。
- **源**: 秋田散人语录, §14
- **关联主张**: C06

## 不对称战役 (Asymmetric Battle)
- **定义**: 只玩信息碾压局，不玩势均力敌的公平对称局。真正的企业家擅长避开竞争或抹杀竞争。
- **源**: 秋田散人语录, §19
- **关联主张**: C09

## 真相通道 (Truth Channel)
- **定义**: 获取一手信息、接近事件真相的管道。包括自有爬虫、信源组合、行业人脉等。建立良好的真相通道是区分普通交易者和职业交易者的关键门槛。
- **源**: 秋田散人语录, §21
- **关联主张**: C03

## 程序即纪律 (Code as Discipline)
- **定义**: 把交易原则和假设写进程序里——每一条if/else代表对市场的一条假设；每一个risk limit代表能承受的最坏情况；每一个alert代表对不确定性的敬畏。
- **源**: 秋田散人语录, §33
- **关联主张**: C10

## 天然企业主 (Born Entrepreneur)
- **定义**: 具有互联网编码能力和传媒影响力的人，即便没有资本，也是天然的企业主。可以用极低成本跨越当地货币与权力，对社会资源进行调度与编码。
- **源**: 秋田散人语录, §31
- **关联主张**: C08, C20

## 高周转率经营 (High-Velocity Operation)
- **定义**: 用高换手率拉大总获利、用快速回血抵消偶发损失。10%利润率周转10次就是100%+（复利）。
- **源**: 秋田散人语录, §16
- **关联主张**: C12

## 锚定资产脱锚 (Pegged Asset Depeg)
- **定义**: 所有带有"价值锚"的代币在局部、短暂、可恢复的定价偏离。"局部且短暂"的脱锚区别于信用彻底崩塌后的永久脱锚。
- **源**: 秋田散人语录, §12
- **关联主张**: C02

## 跨链桥闸口监控 (Cross-Chain Bridge Monitoring)
- **定义**: 跨链桥负责资金闸口、AMM负责流动性——两者是不同的主体。一旦跨链桥停止服务，链上AMM不会自动停盘，形成巨大的定价偏离。
- **源**: 秋田散人语录, §12
- **关联主张**: C05
