[VCP百科全书] 插件说明——TavilySearch（原子切片）

一、插件身份
- name: TavilySearch
- displayName: Tavily 搜索插件
- pluginType: synchronous
- command: TavilySearch

二、核心能力
- 使用 Tavily API 进行高级网络搜索。AI可以指定搜索查询、主题、最大结果数，并可选择包含原始内容、图片链接及描述，以及设定搜索时间范围。
- 提供可调用命令：TavilySearch。

三、占位符与注入点
- systemPromptPlaceholders: 无

四、配置重点
- 优先核对插件目录内的 plugin-manifest.json(.block) 与 config.env.example 必填项。
- 外部API类插件重点检查：密钥、端点、超时、代理与速率限制。

五、排障建议
- 插件无响应：先检查 manifest 启用状态、入口脚本、运行时依赖。
- 结果异常：核对命令名与参数字段，再查看插件目录缓存/日志文件。

六、调用示例
基础搜索:
<<<[TOOL_REQUEST]>>>
tool_name:「始」TavilySearch「末」,
command:「始」TavilySearch「末」,
query:「始」machine learning latest research「末」
<<<[END_TOOL_REQUEST]>>>

高级搜索（包含原始内容和图片）:
<<<[TOOL_REQUEST]>>>
tool_name:「始」TavilySearch「末」,
command:「始」TavilySearch「末」,
query:「始」artificial intelligence trends「末」,
topic:「始」general「末」,
max_results:「始」10「末」,
include_raw_content:「始」true「末」,
include_images:「始」true「末」
<<<[END_TOOL_REQUEST]>>>
