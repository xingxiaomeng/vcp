[VCP百科全书] TableLampRemote / 米家台灯遥控器 - IoT设备控制

一、插件基本信息
1. 插件名称：TableLampRemote / 米家台灯遥控器
2. 插件类型：同步(synchronous)
3. 版本号：1.0.0
4. 所属模块：VCPDistributedServer/Plugin
5. 功能一句话总结：通过mijiaAPI控制米家台灯的开关、亮度、色温

二、功能定位
TableLampRemote是一个IoT设备控制插件，专门用于控制米家台灯Pro。它封装了mijiaAPI的命令行调用，提供简洁的接口让AI能够：
- 获取台灯当前状态（开关、亮度、色温）
- 控制台灯的开关状态
- 调节台灯亮度（1-100）
- 调节台灯色温（2500K-4800K）
- 支持批量操作（同时设置多个属性）

该插件展示了VCP如何与智能家居设备集成，是IoT控制的典型示例。

三、配置参数
该插件无外部配置文件(config.env)，所有配置硬编码在main.py中：

| 配置项 | 值 | 说明 |
|--------|-----|------|
| DEVICE_NAME | "米家台灯Pro" | 设备名称，必须与mijiaAPI中注册的设备名一致 |
| mijiaAPI | python -m mijiaAPI | 依赖的外部Python包命令 |

前置依赖:
1. Python环境已安装mijiaAPI包: pip install mijiaAPI
2. mijiaAPI已完成小米账号配置和设备发现
3. 设备名称"米家台灯Pro"必须在mijiaAPI中正确注册

四、调用命令列表

1. GetLampStatus - 获取台灯状态
   功能: 获取米家台灯Pro的当前状态，一次性返回开关状态、亮度、色温三个信息
   参数: 无
   调用格式:
   <<<[TOOL_REQUEST]>>>
   tool_name:「始」TableLampRemote「末」,
   command:「始」GetLampStatus「末」
   <<<[END_TOOL_REQUEST]>>>

   返回值示例:
   {
     "status": "success",
     "result": "台灯当前状态：开关状态为 on，亮度为 80，色温为 4000。"
   }

2. LampControl - 控制台灯
   功能: 控制米家台灯Pro的各项属性，可一次性设置一个或多个属性
   参数(全部为可选，但至少提供一个):
   - power (string): 控制开关，'True'打开，'False'关闭
   - brightness (integer): 设置亮度，范围1-100
   - color_temperature (integer): 设置色温，范围2500-4800
   调用格式:
   单项控制(仅调亮度):
   <<<[TOOL_REQUEST]>>>
   tool_name:「始」TableLampRemote「末」,
   command:「始」LampControl「末」,
   brightness:「始」80「末」
   <<<[END_TOOL_REQUEST]>>>

   多项控制(批处理):
   <<<[TOOL_REQUEST]>>>
   tool_name:「始」TableLampRemote「末」,
   command:「始」LampControl「末」,
   power:「始」True「末」,
   brightness:「始」100「末」,
   color_temperature:「始」4000「末」
   <<<[END_TOOL_REQUEST]>>>

   仅开关灯:
   <<<[TOOL_REQUEST]>>>
   tool_name:「始」TableLampRemote「末」,
   command:「始」LampControl「末」,
   power:「始」False「末」
   <<<[END_TOOL_REQUEST]>>>

   返回值示例:
   {
     "status": "success",
     "result": "批处理执行完成：\n设置成功：on = True\n设置成功：brightness = 80\n设置成功：color-temperature = 4000"
   }

<<<[END_TOOL_REQUEST]>>>

六、注意事项与限制
1. 设备名称固定：代码中硬编码设备名为"米家台灯Pro"，如需控制其他型号需修改main.py中的DEVICE_NAME常量
2. 依赖mijiaAPI：必须预先安装并配置好mijiaAPI Python包，包括小米账号登录
3. 网络要求：需要能够访问小米IoT服务器
4. 超时时间15000毫秒(15秒)
5. 批处理原子性：LampControl的多项设置是顺序执行的，任一失败会记录但不会中断后续操作
6. 参数类型严格：power必须是字符串'True'/'False'(注意大小写)，brightness和color_temperature必须是整数
7. Windows兼容性：使用shell=True执行命令，已在Windows上测试通过
8. 中文输出：使用utf-8编码确保中文正常显示

七、关联插件
- mijiaAPI：底层依赖的Python库，负责与小米云服务通信
- ScreenPilot：可以截图验证台灯状态变化
- VCPAlarm：可以配合定时任务实现自动开关灯
