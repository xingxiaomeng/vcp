[VCP百科全书] Translatormodules - 翻译模块

一、模块基本信息
1. 模块名称：Translatormodules / 翻译模块
2. 模块类型：前端UI模块
3. 版本号：1.0.0
4. 所属项目：VCPChat
5. 功能一句话总结：提供多语言翻译功能，支持多个AI模型、自定义提示词、流式翻译

二、功能定位
Translatormodules是VCPChat的翻译模块，提供快速、高效的多语言翻译功能。支持多个Gemini模型、多种目标语言、自定义提示词、实时翻译等功能。

三、核心功能
1. 翻译功能：
   - 文本翻译
   - 流式翻译
   - 实时显示翻译结果

2. 模型选择：
   - gemini-2.5-flash-lite-preview（快速）
   - gemini-2.5-flash（均衡）
   - gemini-2.5-pro（质量）

3. 语言支持：
   - 英语
   - 中文
   - 日语
   - 韩语
   - 法语
   - 德语
   - 西班牙语
   - 自定义语言

4. 高级功能：
   - 自定义提示词
   - 翻译中止
   - 复制译文
   - 主题适配

五、主要文件结构
- translator.js：核心逻辑（8KB）
- translator.html：UI布局
- translator.css：样式定义

六、关键API接口
1. window.electronAPI.loadSettings()
   - 获取VCP服务器配置

2. VCP API调用：
   - POST /v1/chat/completions
   - 参数：messages、model、temperature、max_tokens、stream

3. 翻译请求格式：
   {
     messages: [{role: "user", content: "翻译提示词"}],
     model: "gemini-2.5-flash",
     temperature: 0.3,
     max_tokens: 50000,
     stream: false
   }

七、使用场景
1. 快速翻译：翻译文本内容
2. 多语言支持：支持多种语言
3. 自定义翻译：使用自定义提示词

八、与其他模块的关系
- 依赖 Electron API：与主进程通信
- 集成 VCPChat主题系统

九、注意事项
1. 需要配置VCP服务器URL和API Key
2. 支持中止翻译
3. 支持复制译文
4. 自定义提示词可选

Tag: 翻译, 多语言, AI模型, 流式翻译, 自定义提示词
