[VCP百科全书] 插件说明——ZImageGen2（原子切片）

一、插件身份
- name: ZImageGen2
- displayName: Z-Image Base 文生图
- pluginType: synchronous
- command: ZImageGenerate

二、核心能力
- 通过 Hugging Face Spaces API 使用 Z-Image-Base 模型生成高质量图片。支持中英文提示词，28步推理，支持负向提示词和CFG调节。
- 提供可调用命令：ZImageGenerate。

三、占位符与注入点
- systemPromptPlaceholders: 无

四、配置重点
- 优先核对插件目录内的 plugin-manifest.json(.block) 与 config.env.example 必填项。
- 外部API类插件重点检查：密钥、端点、超时、代理与速率限制。

五、排障建议
- 插件无响应：先检查 manifest 启用状态、入口脚本、运行时依赖。
- 结果异常：核对命令名与参数字段，再查看插件目录缓存/日志文件。

六、调用示例
基础生成:
<<<[TOOL_REQUEST]>>>
tool_name:「始」ZImageGen2「末」,
command:「始」ZImageGenerate「末」,
prompt:「始」a beautiful landscape painting「末」
<<<[END_TOOL_REQUEST]>>>

高级参数（包含负向提示词和 CFG 调节）:
<<<[TOOL_REQUEST]>>>
tool_name:「始」ZImageGen2「末」,
command:「始」ZImageGenerate「末」,
prompt:「始」a beautiful landscape painting with mountains and rivers「末」,
negative_prompt:「始」blurry, low quality「末」,
cfg_scale:「始」7.5「末」
<<<[END_TOOL_REQUEST]>>>
