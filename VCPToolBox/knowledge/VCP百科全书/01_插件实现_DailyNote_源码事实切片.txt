[VCP百科全书] 插件实现——DailyNote（源码事实切片）

一、源码锚点
- manifest: Plugin/DailyNote/plugin-manifest.json
- entry: Plugin/DailyNote/dailynote.js
- 主分发: main() -> switch(command) -> create/update
- 关键函数: processTags(), handleCreateCommand(), handleUpdateCommand()

二、行为事实（代码级）
- create 支持参数兼容：maid/maidName/Maid/MAID、Date/dateString、Content/contentText。
- Tag 处理优先级：独立 Tag 字段优先；否则检测 Content 末行 Tag:；两者都无则抛错。
- maid 支持 [文件夹]名字 语法，创建时会拆分 folder 与 actualMaidName。
- update 有安全门槛：target 长度必须 >= 15，否则直接拒绝。

三、边界与错误模式
- create 缺参数返回 Invalid input for create。
- update 非字符串参数返回 Invalid arguments for update。
- 未知命令返回 Unknown command，仅允许 create/update。
