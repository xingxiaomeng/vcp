[VCP百科全书] 插件说明——GrokVideoGen（原子切片）

一、插件身份
- name: GrokVideoGen
- displayName: Grok 视频生成器
- pluginType: synchronous
- command: submit, concat

二、核心能力
- 使用 Grok API 进行视频生成（支持文生视频、图生视频、视频续写、视频拼接）。
- 提供可调用命令：submit、concat。

三、占位符与注入点
- systemPromptPlaceholders: 无

四、配置重点
- 优先核对插件目录内的 plugin-manifest.json(.block) 与 config.env.example 必填项。
- 外部API类插件重点检查：密钥、端点、超时、代理与速率限制。

五、排障建议
- 插件无响应：先检查 manifest 启用状态、入口脚本、运行时依赖。
- 结果异常：核对命令名与参数字段，再查看插件目录缓存/日志文件。

六、调用示例
<<<[TOOL_REQUEST]>>>
tool_name:「始」GrokVideoGen「末」,
command:「始」submit「末」
<<<[END_TOOL_REQUEST]>>>
