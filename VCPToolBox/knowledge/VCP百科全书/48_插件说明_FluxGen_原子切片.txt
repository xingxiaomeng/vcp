[VCP百科全书] 插件说明——FluxGen（原子切片）

一、插件身份
- name: FluxGen
- displayName: Flux 风格图片生成器
- pluginType: synchronous
- command: FluxGenerateImage

二、核心能力
- 通过 SiliconFlow API 使用 FLUX.1-schnell 模型生成具有特定风格的图片。
- 提供可调用命令：FluxGenerateImage。

三、占位符与注入点
- systemPromptPlaceholders: 无

四、配置重点
- 优先核对插件目录内的 plugin-manifest.json(.block) 与 config.env.example 必填项。
- 外部API类插件重点检查：密钥、端点、超时、代理与速率限制。

五、排障建议
- 插件无响应：先检查 manifest 启用状态、入口脚本、运行时依赖。
- 结果异常：核对命令名与参数字段，再查看插件目录缓存/日志文件。

六、调用示例
<<<[TOOL_REQUEST]>>>
tool_name:「始」FluxGen「末」,
command:「始」FluxGenerateImage「末」
<<<[END_TOOL_REQUEST]>>>
